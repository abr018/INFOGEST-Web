# INFOGEST Web

Web application developed using ASP.NET and SQL Server in an academic context.

## 📌 Description
This project is a web application for data management, developed using ASP.NET.

Main features:
- Data management through web pages
- Integration with SQL Server database
- MVC structure (Models, Views, Controllers)
- Input validation

## 🛠️ Technologies Used
- ASP.NET Core
- C#
- SQL Server
- HTML / CSS
